# 🎨 Day 18 – Turtle Graphics & Hirst Painting

## 📖 Overview

On Day 18 of the **100 Days of Python** challenge, I explored Python's built-in **Turtle Graphics** module and learned how to create graphical patterns using Object-Oriented Programming.

Throughout this project, I drew various geometric shapes, implemented a random walk with randomly changing colors and directions, created a spirograph using circles, and recreated a simplified version of Damien Hirst's famous dot painting using colorful dots arranged in a grid.
