# 🃏 Day 11 - Blackjack Game

## 📖 Overview

Welcome to **Day 11** of my **100 Days of Python** challenge!

In this project, I built a terminal-based **Blackjack** game in Python. The game follows the basic rules of Blackjack, allowing the player to compete against a computer dealer. Throughout this project, I focused on organizing my code into reusable functions and implementing the game's core logic from scratch.

## ▶️ How to Run

1. Clone this repository.
2. Navigate to the **Day-11** folder.
3. Make sure Python is installed.


## 📚 What I Learned

During this project, I learned how to:

- Build a complete terminal game using Python.
- Break a large problem into smaller reusable functions.
- Calculate scores while handling special Blackjack rules.
- Handle the Ace card (11 or 1) dynamically.
- Detect a Blackjack hand.
- Compare player and dealer scores correctly.
- Make the dealer draw cards until reaching at least 17.
- Restart the game without restarting the program.
- Improve and refactor code through debugging and testing.

## 🚀 Features

- 🎴 Random card dealing
- 🂡 Blackjack detection
- ♠️ Automatic Ace value adjustment (11 → 1)
- 🤖 Dealer AI that follows Blackjack rules
- 🔄 Play again without restarting the program
- 🏆 Win, Lose, Draw, Blackjack, and Bust detection
