# Day 2 - Tip Calculator

## Overview

This is my second project from the **100 Days of Code: The Complete Python Pro Bootcamp**.

The program calculates how much each person should pay when splitting a bill, including a user-selected tip percentage.

## Concepts Practiced

* Printing output using `print()`
* Taking user input with `input()`
* Variables
* Data type conversion (`int()` and `float()`)
* Mathematical operations
* Rounding numbers using `round()`
* Formatted string output (f-strings)

## How to Run

1. Clone this repository.
2. Navigate to the project folder.
3. Run the program:

```bash
py main.py
```

## Example

```text
Welcome to the Tip Calculator!
What was the total bill? $150
What percentage tip would you like to give? 10, 12, or 15? 12
How many people to split the bill? 5

Each person should pay: $33.60
```

## What I Learned

* How to perform calculations using user input.
* The importance of converting strings to numeric data types before performing arithmetic.
* How to round decimal values for user-friendly output.
* How to display formatted results using Python f-strings.
