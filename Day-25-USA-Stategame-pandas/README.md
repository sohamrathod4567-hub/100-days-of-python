# 🐼 Day 25 - U.S. States Game with Pandas

## 📌 Overview

On **Day 25** of the **100 Days of Python** course, I learned how to use the **Pandas** library for working with structured data. I explored how to read, manipulate, and write CSV files, making data handling much simpler than using plain Python.

To apply these concepts, I built the **U.S. States Guessing Game**, where players guess the names of U.S. states. Correct guesses are displayed on the map, and any missed states are saved to a separate CSV file for future practice.

---

## 🛠️ Concepts Practiced

- Importing and using the **Pandas** library
- Reading CSV files with `pd.read_csv()`
- Creating and working with **DataFrames**
- Accessing rows and columns
- Filtering data using conditions
- Converting DataFrame columns into Python lists
- Writing data to CSV files with `to_csv()`
- Integrating Pandas with the Turtle graphics module
- Working with user input and game logic

---

## ▶️ How to Run

1. Make sure Python is installed.
2. Install Pandas if you haven't already:

```bash
pip install pandas
```

3. Clone this repository:

```bash
git clone https://github.com/your-username/100-Days-of-Python.git
```

4. Navigate to the Day 25 folder:

```bash
cd Day-25
```

5. Run the program:

```bash

```

```text

✔ Correct!
```
