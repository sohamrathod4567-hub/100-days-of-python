# Day 5 - Highest Score & Password Generator

## Overview

This is my fifth project from the **100 Days of Code: The Complete Python Pro Bootcamp**.

Day 5 focused on mastering Python loops and building programs that process collections of data. I completed two mini-projects:

1. **Highest Score** – A program that finds the highest score from a list of student scores without using Python's built-in `max()` function.
2. **Password Generator** – A program that generates a secure random password based on the number of letters, symbols, and numbers selected by the user.

## Concepts Practiced

* `for` loops
* Python Lists
* Variables
* Conditional statements (`if`, `else`)
* Comparison operators
* The `random` module
* List methods (`append()`, `shuffle()`)
* String manipulation
* User input using `input()`
