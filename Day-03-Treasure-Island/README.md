# Day 3 - Treasure Island & Python Pizza

## Overview

This is my third project from the **100 Days of Code: The Complete Python Pro Bootcamp**.

Day 3 focused on using conditional statements to build interactive Python programs. I completed two mini-projects:

1. **Treasure Island** – A text-based adventure game where the player's choices determine the outcome.
2. **Python Pizza** – A pizza ordering program that calculates the final bill based on the user's selected pizza size and optional toppings.

## Concepts Practiced

* Conditional statements (`if`, `elif`, `else`)
* Nested conditions
* Logical operators (`and`, `or`)
* Comparison operators
* User input using `input()`
* Variables
* Basic arithmetic operations
* Decision-making in Python

## How to Run

1. Clone this repository.
2. Open the project folder.
3. Run either project:

```bash
py treasure_island.py
```

or

```bash
py python_pizza.py
```

> If your filenames are different (for example, `main.py`), replace the commands above with the correct filename.
