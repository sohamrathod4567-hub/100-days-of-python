# Day 27 - Miles to Kilometers Converter 🛣️➡️📏

## Overview

On Day 27, I learned how to build desktop applications using **Tkinter**, Python's standard GUI (Graphical User Interface) library. Instead of creating programs that only run in the terminal, I built my first GUI application—a **Miles to Kilometers Converter**.

This project introduced me to creating windows, labels, buttons, input fields, and arranging widgets using Tkinter's layout managers.

---

## Concepts Practiced

* Introduction to Tkinter
* Creating a GUI window
* Using `Label` widgets
* Using `Entry` widgets for user input
* Creating `Button` widgets
* Handling button click events with `command`
* Retrieving user input using `.get()`
* Updating label text dynamically with `.config()`
* Organizing widgets using the `grid()` layout manager
* Basic unit conversion logic

---

## How to Run

1. Make sure Python is installed.
2. Clone this repository:

   ```bash
   git clone https://github.com/your-username/100-Days-of-Python.git
   ```
3. Navigate to the Day 27 folder:

   ```bash
   cd Day-27
   ```
4. Run the program:

   ```bash
   python main.py
   ```

---

## Example

```
Input:
10

Output:
16.09 Km
```

The application allows the user to enter a distance in miles and instantly converts it to kilometers when the **Calculate** button is clicked.
