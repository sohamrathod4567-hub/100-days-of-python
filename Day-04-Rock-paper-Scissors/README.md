# Day 4 - Banker Roulette & Rock Paper Scissors

## Overview

This is my fourth project from the **100 Days of Code: The Complete Python Pro Bootcamp**.

Day 4 focused on working with Python lists and the `random` module to build interactive programs. I completed two mini-projects:

1. **Banker Roulette** – A program that randomly selects one person from a group to pay the bill.
2. **Rock Paper Scissors** – A classic game where the user plays against the computer, with the winner determined based on the game's rules.

## Concepts Practiced

* Python Lists
* List indexing
* The `random` module
* Random selection using `random.choice()`
* User input using `input()`
* Conditional statements (`if`, `elif`, `else`)
* Nested conditions
* Variables
* Basic game logic

## How to Run

1. Clone this repository.
2. Open the project folder.
3. Run either project:

```bash
py banker_roulette.py
```

or

```bash
py rock_paper_scissors.py
```

> If your filenames are different (for example, `main.py`), replace the commands above with the correct filename.
