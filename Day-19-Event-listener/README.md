# 🐍 Day 19 - Turtle Race & Higher Order Functions

## 📌 Overview

On **Day 19**, I explored **Higher Order Functions**, **Event Listeners**, and **Object State** using Python's built-in `turtle` module.

I built two fun projects:

* 🎨 **Etch-A-Sketch** – Control a turtle using keyboard inputs to draw on the screen.
* 🐢 **Turtle Race** – A colorful racing game where the user predicts which turtle will win.

This day focused on making programs interactive through keyboard events and random movement.

---

## 🚀 Concepts Practiced

* Object State
* Higher Order Functions
* Event Listeners
* Keyboard Bindings
* `turtle.Screen()`
* `screen.listen()`
* `screen.onkey()`
* Random Number Generation
* Lists of Objects
* Loops and Conditional Statements
* User Input
* OOP with Turtle Objects
