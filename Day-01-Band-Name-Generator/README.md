# Day 1 - Band Name Generator

## Overview

This is my first project from the **100 Days of Code: The Complete Python Pro Bootcamp**.

The program generates a fun band name by combining the name of the city where the user was born with the name of their pet.

## Concepts Practiced

* Printing output using `print()`
* Taking user input with `input()`
* Variables
* String concatenation
* Basic Python syntax

## How to Run

1. Clone this repository.
2. Navigate to the project folder.
3. Run the program:

```bash
py main.py
```

## Example

```text
Welcome to the Band Name Generator
What's the name of the city you were born in?
London
What is your pet's name?
Max

Your band name can be: LondonMax
```

## What I Learned

* How to interact with users through the terminal.
* How to store user input in variables.
* How to combine strings to produce dynamic output.
