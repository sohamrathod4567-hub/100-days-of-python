# Day 26 - NATO Phonetic Alphabet Generator

## 📌 Overview

On Day 26 of the **100 Days of Python** challenge, I learned how to use **List Comprehensions** and **Dictionary Comprehensions** to write cleaner and more Pythonic code.

The main project for the day was a **NATO Phonetic Alphabet Generator**, where the user enters a name or word, and the program converts each letter into its corresponding NATO phonetic code word.

Example:

```text
Input: SOHAM

Output:
['Sierra', 'Oscar', 'Hotel', 'Alfa', 'Mike']
```

I also completed a data overlap exercise that involved reading data from multiple files and finding common values using list comprehensions.
